# Gym Management System – Flask Project

This project is a basic Gym Management System website developed using the Flask
framework. It demonstrates backend routing, template rendering, and static file
handling using Flask, along with frontend design using HTML and CSS.

## Purpose
The purpose of this project is to understand Flask fundamentals such as route
management, dynamic templates, and the use of static files. This project was
created as part of the Introduction to Computing lab.

## Features
- Home page with welcome message and gym highlights  
- About Us page describing the gym’s mission and facilities  
- Membership page showcasing membership plans and services  

## Technologies Used
Python, Flask, HTML, CSS
